날짜: 2025-08-03

🚨 중요: Position Mode 변경 필요!

현재 설정:
❌ Position Mode: Hedge Mode (양방향 포지션)

필요한 설정:
✅ Position Mode: One-way Mode (단방향 포지션)

================================================================

📋 권장 설정값:

1. Position Mode
   현재: Hedge Mode
   → 변경: One-way Mode ⭐ 가장 중요!

2. Unrealized PnL calculation  
   현재: Market price ✅ (유지)

3. Price protection
   현재: Disabled ✅ (유지)

4. ADL ranking
   현재: Hide ✅ (유지)

5. TP/SL trigger price
   현재: Market price ✅ (유지)

================================================================

🔧 설정 변경 방법:

1. Bitget 웹사이트 로그인
2. 선물 거래 페이지로 이동
3. 우측 상단 설정 아이콘 클릭
4. "Position Mode" 찾기
5. "Hedge Mode" → "One-way Mode" 변경
6. 확인 버튼 클릭

⚠️ 주의: 기존 포지션이 있으면 변경할 수 없습니다.
모든 포지션을 종료한 후 변경하세요.

1. 주문 실행 오류 완전 해결

3. 시스템 안정성 향상

📞 변경 완료 후 확인 방법:
